Natural Language Scripting

Describe what you want to extract in plain English. Our LLM generates a Playwright script, tests it live, and iterates until it works.

Auto-Heal Selectors

When a site changes its DOM structure, ScriptFlow automatically detects broken selectors and regenerates them using AI — no manual intervention.

📐

Schema Inference

AI analyzes the page structure and infers the ideal data schema. Outputs are type-safe JSON, CSV, or Parquet — your choice.

🧪

AI Quality Checks

After each run, an AI layer validates data freshness, completeness, and type consistency — flagging anomalies before they reach your warehouse.

Example

Describe it. Ship it.

TypeScript · ScriptFlow SDK 
// Describe your target in natural language
import { ScriptFlow } from '@scriptflow/sdk';

const sf = new ScriptFlow({ apiKey: process.env.SF_API_KEY });

// AI generates and runs the script automatically
const result = await sf.ai.extract({
  url: 'https://example.com/products',
  prompt: 'Extract all product names, prices, ratings, and stock status',
  schema: {
    name:   'string',
    price:  'number',
    rating: 'number',
    inStock:'boolean',
  },
  options: {
    paginate: true,
    maxPages: 10,
    autoHeal: true,
  }
});

console.log(result.data); // Typed, validated JSON array

Concurrent Execution

Run thousands of scraping jobs in parallel across geographically distributed workers. Full isolation prevents cross-job contamination.

50+ Proxy Regions

Structured Output Formats

Smart Retries & Fallbacks

Exponential backoff, configurable retry strategies, and automatic proxy failover ensure every job completes — even on flaky sites.

Full Playwright API

Capture full-page screenshots or generate PDFs on any URL at any viewport. Perfect for visual regression testing and archiving.
