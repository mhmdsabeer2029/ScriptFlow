## 🎯 Project Overview

ScriptFlow serves as a masterclass in modern, dependency-free frontend development. It avoids heavy frameworks in favor of highly optimized vanilla web technologies. The aesthetic revolves around a sophisticated **Dark Navy (`#060d1b`)** and **Gold (`#c8a030`)** color palette, utilizing glassmorphism, layered drop-shadows, and glowing borders to convey premium enterprise quality.

---

## ✨ Key Features & Interactions

We have packed this project with custom-built, lightweight interactive components:

- **🌌 Interactive Particle Canvas:** A custom HTML5 Canvas on the hero section that renders a connected particle network, dynamically reacting to mouse movements.
- **🖱️ Custom Smart Cursor:** A tailored cursor with an outer ring that smoothly follows the user and expands when hovering over interactive elements.
- **🎭 Seamless Page Transitions:** Smooth fade-in/fade-out overlays when navigating between pages to eliminate abrupt browser flashes.
- **📜 Scroll-Driven Reveals:** Elements elegantly animate into the viewport using the highly performant `IntersectionObserver` API.
- **📊 Dynamic Progress Bar:** A fixed gold progress bar at the top of the viewport that tracks reading depth in real-time.
- **🔔 Custom Toast Notifications:** A non-intrusive, animated alert system for form submissions and user feedback.

## 🗺️ Page Architecture

The platform consists of four interconnected, responsive pages:

### 🏠 1. Home (`index.html`)

The main entry point designed for maximum conversion.

- **Dynamic Hero:** Features the interactive canvas background and a typewriter animation cycling through core value propositions.
- **Infinite Trust Marquee:** A seamless, looping ticker of trusted client logos.
- **Animated Number Counters:** Statistics that smoothly count up to their target values as they enter the screen.
- **Testimonial Slider:** A touch-enabled, auto-playing carousel showcasing social proof.

### ⚙️ 2. Features (`features.html`)

A deep dive into technical capabilities.

- **Dynamic Tab Interface:** Seamlessly switch between AI, Scraping, Browser, and Integration features without page reloads.
- **Interactive Code Showcase:** A stylized IDE-like code block with a functional "Copy to Clipboard" button.
- **Responsive Comparison Table:** A clean data table benchmarking ScriptFlow against competitors.

### 💳 3. Pricing (`pricing.html`)

A transparent, trust-building pricing breakdown.

- **Animated Billing Toggle:** A smooth switch between "Monthly" and "Annual" billing that dynamically recalculates and animates the displayed prices.
- **Highlighted Tier Cards:** The "Pro" tier features layered glowing shadows and hover scaling to draw the eye.
- **Expandable FAQ:** A CSS-transition-powered accordion for common billing questions.

### 📞 4. Contact (`contact.html`)

A dedicated support and sales inquiry portal.

- **Real-time Form Validation:** Actively validates email formatting and required fields as the user types.
- **Visual Error States:** Input fields glow gold on focus and turn red with helper text if validation fails.
- **Mock Submission Workflow:** Simulates a server request, triggers a success Toast notification, and resets the form.

## 🛠️ Technology Stack

We built this project to be inherently fast, accessible, and easy to host anywhere.

- **Markup:** Semantic HTML5
- **Styling:** CSS3 (Custom Properties, Grid/Flexbox, `clamp()` fluid typography, complex `@keyframes`)
- **Logic:** Vanilla JavaScript (ES6+, Functional Modules, `IntersectionObserver`, `requestAnimationFrame`)
- **Assets:** Inline SVGs for infinite scalability and zero HTTP requests.
